# Hyper-Kinetic Brutalism

A Pen created on CodePen.

Original URL: [https://codepen.io/aleksa-rakocevic/pen/JoKKjwd](https://codepen.io/aleksa-rakocevic/pen/JoKKjwd).

An exploration of raw interaction physics and aggressive aesthetics. This showcase features: • Morphing 3D Responsive Navbar (Dock to Float transformation) • Inertial Scroll Skew (Content distorts based on scroll velocity) • Magnetic Cursor & Element Snapping (Lerp physics) • RGB Split Glitch & Hacker Text Decoding • SVG Noise Overlay & Mix-Blend-Mode Compositing

Built entirely with Vanilla JS and CSS variables. No external libraries.
